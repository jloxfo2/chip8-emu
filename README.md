chip8-emu is an aptly named CHIP-8 emulator written in C.

Compile with:  gcc emulator.c cpu.c -lGL -lGLU -lglut -o chip8


Keyboard to Hexpad mapping:
 _______              _______
|1|2|3|4|            |0|1|2|3|
 |Q|W|E|R|    ====>   |4|5|6|7| 
  |A|S|D|F|            |8|9|A|B|
